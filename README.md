## 💻 Como baixar o projeto 
-Requisitos 
- Node + NPM;
- Yarn;
- Visual Studio Code e configurações.

```bash
#Clonando o repositorio 
$ git clone https://github.com/SamiraFreitas/Nlw
#Entrar no diretorio 
$ cd Nlw
$ cd letmeask
# Instalar as dependências
$ npm install || yarn install
#Rode o projeto 
$ yarn start
#Acesse o site 
$ localhost:5000/
```
